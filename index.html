Wedding Planner

This site was created from scratch using HTML and JavaScript, with added CSS styling. I made a formula to distribute a set number of guests among a set number of tables as equally as possible. I then made it so users could try out different combinations and have the page keep track of each output generated.

Dice Game

I improved upon a prexisting file to make it so the user roll multiple times and have the website keep track of the balance. Using JavaScript, I was also able to slow the animation of the dice roll, allowing users to keep up with each roll. I then used CSS to alter the style of the page.

FIGMA

I used Figma to create the layout of a period tracker app, creating a logo, icons, and a color scheme. The flexibility of Figma allowed me to prototype and connect frames to grasp the full vision of this app.
